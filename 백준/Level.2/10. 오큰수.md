# 오큰수

### [ 문제 정보 ]
> **백준**: https://www.acmicpc.net/problem/17298
> 
> **난이도**: 2
>
> **유형**: #구현


### [ 문제 풀이 ]
```C++
#include <iostream>
#include <cstring>
#include <stack>
using namespace std;

int arr[1000001], brr[1000001];
int n;
stack<int> st;
int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL); cout.tie(NULL);

    memset(brr, -1, sizeof(brr));

    cin >> n;
    for (int i = 0; i < n; i++) {
        cin >> arr[i];
    }

    for (int i = 0; i < n; i++) {
        while (st.size() && arr[st.top()] < arr[i]) {
            brr[st.top()] = arr[i];
            st.pop();
        }
        st.push(i);
    }

    for (int i = 0; i < n; i++) {
        cout << brr[i] << ' ';
    }
    return 0;
}
```
