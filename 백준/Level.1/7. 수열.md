# 수열

### [ 문제 정보 ]
> **백준**: https://www.acmicpc.net/problem/2559
> 
> **난이도**: 1
>
> **유형**: #구현


### [ 문제 풀이 ]
```C++
#include <iostream>
using namespace std;

int n, k, ret = -987654321;
int arr[100004];
int pSum[100004];
int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL); cout.tie(NULL);

    cin >> n >> k;
    for (int i = 1; i <= n; i++) {
        cin >> arr[i];
        pSum[i] = pSum[i - 1] + arr[i];
    }

    for (int i = k; i <= n; i++) {
        ret = max(ret, pSum[i] - pSum[i - k]);
    }

    cout << ret << '\n';
    return 0;
}
```
